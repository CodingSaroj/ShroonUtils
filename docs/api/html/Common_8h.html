Detailed Description

This file defines special macros which will be used instead of normal standard library functions.

This allows the possibility of a custom implementation of the standard functions (Useful for using this library without linking the standard library).

These macros use prefix SHRN instead of SUTL which allows the custom functions to be defined only once for all Shroon projects.
